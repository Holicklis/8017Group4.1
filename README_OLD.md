# STAT 8017B Project 4 — Financial Analysis Chatbot

**Group 4.1**

## Quick Start

### 1. Clone the repo
```bash
git clone <repo-url>
cd 8017project
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Get the data

Data is shared via OneDrive (too large for GitHub — ~3.3 GB raw).
Copy the entire `data/` folder into the project root so the structure is:
```
8017project/
├── data/
│   ├── alpha-insights/
│   ├── complaints/
│   ├── finance-data/
│   ├── financial-news/
│   ├── financial-qa/
│   ├── finsen/
│   ├── phrasebank/
│   ├── sp500-etf-crypto/
│   └── processed/          (generated by notebook 01)
├── notebooks/
├── app/
...
```

### 4. Install Ollama + Phi-3.5

1. Download Ollama from https://ollama.com/download
2. Install and open a new terminal
3. Run: `ollama pull phi3.5`

### 5. Run notebooks in order

```
01_preprocessing.ipynb   → Cleans all 8 datasets → data/processed/
02_classification.ipynb  → Trains classifiers → models/
03_regression.ipynb      → Trains regressors → models/
04_unsupervised.ipynb    → PCA, K-Means, Apriori, LSA analysis
05_chatbot.ipynb         → Chatbot pipeline + end-to-end test → models/
```

### 6. Launch the chatbot app

```bash
streamlit run app/chatbot_app.py
```
